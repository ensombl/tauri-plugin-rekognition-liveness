## Default Permission

Default permissions exposing the AWS Face Liveness detection command.

#### Granted Permissions

It allows access to the `detect_liveness` command — opens the native
Amplify Liveness UI on Android/iOS and resolves with a single status object.

#### This default permission set includes the following:

- `allow-detect-liveness`
